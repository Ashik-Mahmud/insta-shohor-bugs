What's difference between LocalStorage & SessionStorage?

LocalStorage & SessionStorage basically Web API that means it's built-in side of browser. Those are works like as storages to store bit of data to tracking & remaining users. It's almost same then it's has a bit of difference which one is-

LocalStorage SessionStorage
LocalStorage data storing capacity limit almost 5-10 MB SessionStorage data storing capacity limit 5MB
Data never remove unless you remove manually or using javascript. Data remove if user close the particular browser or Tab.
LocalStorage use for storing data long time SessionStorage use for storing data temporary just use until browser open
LocalStorage can't send a http request into server. SessionStorage can't send a http request into server as well.
I think localStorage is less secure(e.g - another person can access your data later on) I think sessionStorage is more secure
